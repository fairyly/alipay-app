# alipay-app

支付宝小程序-ant app

>公测期间尚未开放个人支付宝账户

## 准备

- 注册支付宝企业账户
- 进入开放平台管理中心，选择“开发者中心／小程序”，开始【创建小程序】
- 用户授权：https://docs.alipay.com/mini/introduce/authcode
- 服务端SDK：https://docs.open.alipay.com/54/103419/
- 下载蚂蚁开发者工具：https://docs.alipay.com/mini/ide/download
